# Hey, I'm Josh Simon 👋

**Software Engineer · Distributed Systems · RF Security**

---

### ⚙️ Software Engineering

I build and architect core backend systems using Java and Spring Boot — distributed microservice architectures, event-driven pipelines, inter-service messaging, and data systems that scale. I care about how systems are designed, how they break, and how they recover. On the mobile side I do core Android development in Java and Kotlin, from the ground up.

---

### 📡 RF & Radio Security

Radio frequency engineering and hardware is where my deepest interest lies. I reverse engineer communication protocols, analyse what's transmitting across the spectrum, and build SDR pipelines in GNU Radio and C++. I design and test antennas, explore wireless security, and dig into RF infrastructure to understand how it breaks and how it can be hardened. Working toward CEH/OSCP.
